# Calculator

A simple, interactive web-based calculator built with HTML, CSS, and JavaScript.

## Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **Clear Functions**: 
  - `AC` (All Clear) - Resets the entire calculator
  - `Clear` - Clears the current input while preserving the operator
- **Real-time Display**: Shows numbers as you type and results after calculation
- **Clean UI**: Responsive calculator interface with intuitive button layout

## Project Structure

- `index.html` - HTML structure with calculator buttons and display
- `index.js` - JavaScript functionality for calculator operations
- `styles.css` - CSS styling for the calculator layout
- `plan.txt` - Project planning notes

## How to Use

1. Open `index.html` in your web browser
2. Click number buttons to enter your first number
3. Click an operator button (`+`, `-`, `x`, `/`)
4. Click number buttons to enter your second number
5. Click `=` to see the result
6. Use `Clear` to remove the current input or `AC` to reset everything

## Technical Details

The calculator uses:
- **Vanilla JavaScript** for all functionality (no frameworks)
- **Event listeners** on number and operator buttons
- **Switch statement** for operation handling
- **CSS Flexbox** for responsive layout

## Supported Operations

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`x`)
- Division (`/`)
